Laboratory

Tutorial’s laboratory is based on a set of technologies and resources which will allow you to understand what Red Hat Service Mesh, Istio Control Plane and Envoy roles are:

  • Red Hat Openshift Container Platform

  • Red Hat Service Mesh

  • A GitHub Repositories

Instructor provides students Openshift credentials at the beginning of this tutorial in order to access the laboratory.

  • Openshift Cluster URL (E.g. https://console-openshift-console.apps.acidonpe.sandbox507.opentlc.com)

  • Username (E.g. user1)

  • Password (E.g. password)

  • Namespace (E.g. ns_user1)

  • Openshift Apps Domain (E.g. apps.acidonpe.sandbox507.opentlc.com)

Please, review the previous settings with your instructor and connect to the cluster executing next command:

oc login -u <user> -p <pass> <ocp_cluster_url>
Login successful.

You have access to 65 projects, the list has been suppressed. You can list all projects with 'oc projects'

Using project "default".

In addition, access to <ocp_cluster_url> via browser in order to access Openshift Console.
